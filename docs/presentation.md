## 📑 شرائح العرض (Slide Deck Outline)

### الشريحة 1: الغلاف
- **DriverTRACK**: منصة تتبع حضور السائقين
- شعار NAVEX + DriverTRACK
- "حل ذكي لإدارة الحضور عبر رمز QR"

### الشريحة 2: المشكلة والحل
| المشكلة | الحل |
|---------|------|
| تسجيل حضور يدوي عرضة للأخطاء | مسح QR آلي وموثق |
| صعوبة تتبع التأخير والغياب | كشف تلقائي مع غرامات |
| عدم وجود تقارير فورية | لوحة تحكم وإحصائيات مباشرة |
| إدارة عدة محطات وفرق | نظام متكامل متعدد الأدوار |

### الشريحة 3: الميزات الرئيسية
1. 🔐 تسجيل دخول بصلاحيات (مدير / مشغل / سائق)
2. 📱 تطبيق سائق (Capacitor APK + iOS)
3. 📷 مسح QR للحضور مع التوقيع الرقمي HMAC
4. ⏰ كشف التأخير حسب فترات العمل القابلة للتعديل
5. 💰 غرامات مالية تلقائية (150 دج)
6. 📊 لوحة تحكم برسوم بيانية ومؤشرات أداء
7. 🔔 مركز إشعارات فوري للمتأخرين
8. 📄 تصدير Excel + PDF

### الشريحة 5: العرض المباشر (Live Demo)
- تسجيل الدخول كمدير
- استعراض لوحة التحكم
- تصفية التاريخ وعرض الإحصائيات
- فتح إدارة السائقين
- مسح QR تجريبي
- عرض الإشعارات

### الشريحة 6: التقارير
- Excel: تقارير حضور قابلة للتصدير
- PDF: تقارير غرامات فردية لكل سائق
- تصفية حسب التاريخ والمحطة والسائق

### الشريحة 7: الأمان والموثوقية
- JWT مع صلاحية 365 يوماً
- توقيع QR رقمي (HMAC-SHA256)
- حماية ضد تسجيل الحضور المكرر
- تحديل أوقات القطع في أي وقت (بدون برمجة)
